# MD Editor

**MD Editor** — Markdown-редактор для десктопа и браузера (v1.3.0).

Стек: **Vite 8**, **Tauri 2**, vanilla JS, **marked**, **highlight.js**, **docx**, **html2pdf.js**.

Редактор с живым HTML-предпросмотром, вкладками, оглавлением, поиском/заменой, настраиваемым интерфейсом, экспортом в HTML/DOCX/PDF и нативной работой с файлами в десктоп-сборке.

## Возможности

### Редактор и предпросмотр

- Двухпанельный интерфейс: исходный Markdown и HTML-предпросмотр
- Режимы отображения: обе панели / только редактор / только превью
- Полный экран (`F11`)
- Изменяемая граница между редактором и предпросмотром (перетаскивание, клавиатура, двойной щелчок — 50/50)
- Живой рендер через [marked](https://github.com/markedjs/marked)
- Нумерация строк и подсветка текущей строки
- Статус-бар: число слов, символов, позиция курсора (строка / колонка)
- Копирование содержимого блоков кода в превью (кнопка **Copy**)
- **Подсветка синтаксиса** в блоках кода (highlight.js, тема github-dark)
- **Метка языка** в левом верхнем углу блоков кода (bash, python, js и др.)
- Тёмная и светлая темы (сохраняются в `localStorage`)
- **Автосохранение** каждые 3 секунды + восстановление сессии после перезагрузки страницы

### Вкладки

- Несколько документов одновременно
- **Создать** / **Открыть** — открывают отдельную вкладку
- Переключение вкладок с сохранением содержимого, курсора и прокрутки
- **Ctrl+Tab** — быстрое переключение по последнему использованию (MRU)
- Отдельная история undo/redo на каждую вкладку
- Индикатор несохранённых изменений (точка на вкладке)
- Закрытие вкладки с подтверждением, если есть несохранённые правки

### Файлы и меню

**Строка 1 шапки:** логотип · **Файл ▾** / **Настройки ▾** · Вид / Оглавление / Полный экран / Тема  

**Строка 2:** панель инструментов (настраиваемая)

#### Меню «Файл ▾»

| Действие | Поведение |
|----------|-----------|
| **Создать** (`Ctrl+N` / `Ctrl+T`) | Новая вкладка |
| **Открыть** | Диалог выбора `.md` / `.markdown` / `.mdown` / `.txt` |
| **Сохранить** (`Ctrl+S`) | Тихая перезапись уже открытого файла (без диалога) |
| **Сохранить как…** (`Ctrl+Shift+S`) | Диалог выбора пути / имени |
| **Экспорт HTML** | Сохранение как HTML-документ |
| **Экспорт DOCX** | Сохранение как Word-документ |
| **Экспорт PDF** | Сохранение как PDF (A4) |
| Новый документ → Сохранить | Сначала «Сохранить как…», далее — тихая запись |

- **Tauri (десктоп):** нативные диалоги и запись на диск (`src-tauri`, rfd)
- **Браузер (Chrome / Edge):** [File System Access API](https://developer.mozilla.org/en-US/docs/Web/API/File_System_Access_API)
- **Fallback:** download, если нет привязки к файлу или API недоступен

#### Меню «Настройки ▾»

| Пункт | Описание |
|-------|----------|
| **Интерфейс** | Шрифты, цвета, фоны, курсор, контраст кода |
| **Панель инструментов** | Перестановка кнопок и разделителей |
| **О программе** | Компактное окно: название, версия, copyright, GitHub |

### Настройки интерфейса

Отдельно для **редактора** и **предпросмотра**:

- семейство шрифта (UI, mono, Segoe, Arial, Georgia, Consolas и др.)
- размер (10–28 px)
- цвет текста и фон («как в теме» или свой цвет)

Дополнительно:

- **Контраст кода** в превью: мягкий / обычный / высокий / максимальный
- **Курсор редактора:** цвет, ширина/форма (линия, блок, подчёркивание), моргание (да / медленно / нет)

Хранятся в `localStorage` (`md-editor-interface`).

### Панель инструментов

- **Настройки → Панель инструментов** — режим настройки
- Перетаскивание кнопок и разделителей (Pointer Events — работает в Chrome и WebView2)
- **+ Разделитель**, двойной клик по разделителю — удалить
- **Сбросить** / **Готово** (или `Esc`)
- Раскладка в `localStorage` (`md-editor-toolbar-layout`)

### Таблицы

- GFM-таблицы в Markdown рендерятся в превью
- Кнопка **Copy** у каждой таблицы: **HTML + TSV** (Word / Excel)
- Экспорт в **DOCX** с настоящими таблицами Word

### Экспорт

- **HTML** — полноценный HTML-документ
- **DOCX** — Word (`docx`): заголовки, списки, код, цитаты, ссылки, таблицы, индексы
- **PDF** — A4 (`html2pdf.js`); в десктопе локальные картинки встраиваются с диска

### Форматирование

- Отмена / повтор, вырезать / копировать / вставить
- **Жирный**, *курсив*, ~~зачёркивание~~, `инлайн-код`, ==выделение==
- Верхний / нижний индекс (`<sup>` / `<sub>`)
- Заголовки H1–H4, списки, задачи, цитата, блок кода, HR
- Изображение (data URL) и ссылка
- Автозакрытие скобок и кавычек
- Подсветка активных кнопок и тогл форматирования

### Контекстное меню

Правый клик по выделенному тексту: inline, индексы, списки, блоки, медиа.

### Оглавление (TOC)

- Боковая панель H1–H4, переход по клику, подсветка текущего раздела
- **Вставить** / **Обновить** блок `<!-- TOC -->` … `<!-- /TOC -->` в документе

### Поиск и замена

- Найти далее, заменить, заменить все, счётчик `N / M`

### Локальные изображения (десктоп)

- Относительные пути `![](photo.png)` резолвятся относительно каталога открытого `.md`
- Чтение через Rust IPC (`read_local_image_data_url`) → data URL в превью
- Документ нужно **открыть с диска** (`Файл → Открыть`)

### Горячие клавиши

Работают по `event.code` (не зависят от раскладки). На macOS — `Cmd` вместо `Ctrl`.

| Клавиши | Действие |
|---------|----------|
| `Ctrl+B` | Жирный |
| `Ctrl+I` | Курсив |
| `Ctrl+Shift+X` | Зачёркивание |
| `Ctrl+Shift+H` | Подсветка |
| `Alt+1`..`Alt+4` | H1–H4 |
| `Ctrl+S` / `Ctrl+Shift+S` | Сохранить / Сохранить как… |
| `Ctrl+N` / `Ctrl+T` | Новая вкладка |
| `Ctrl+W` | Закрыть вкладку |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Вкладки (MRU) |
| `Ctrl+F` / `Ctrl+H` | Поиск / замена |
| `F11` | Полный экран |
| `Ctrl+Z` / `Ctrl+Y` | Отменить / Повторить |

В браузере перехватываются `Ctrl+N` и `Ctrl+Shift+T`.

## Заметки

- Индексы — HTML `<sup>` / `<sub>` (превью и DOCX).
- Локальные картинки в десктопе: откройте `.md` с диска; пути с пробелами поддерживаются.
- Кнопка «Вставить изображение» пишет data URL (файл может стать большим).
- Горячие клавиши по физическим клавишам — работают при любой раскладке.
- Настройки интерфейса и панели не сбрасываются при смене темы (цвета «как в теме» следуют теме).
