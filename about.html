Возможности

Редактор и предпросмотр

  • Двухпанельный интерфейс: исходный Markdown и HTML-предпросмотр
  • Режимы отображения: обе панели / только редактор / только превью
  • Полный экран (F11)
  • Изменяемая граница между панелями (перетаскивание, клавиатура, двойной щелчок — 50/50)
  • Живой рендер через marked, подсветка кода (highlight.js)
  • Нумерация строк, статус-бар, копирование кода и таблиц в превью
  • Тёмная и светлая темы
  • Автосохранение каждые 3 секунды + восстановление сессии

Вкладки

  • Несколько документов, MRU-переключение Ctrl+Tab
  • Отдельный undo/redo, индикатор несохранённых правок
  • Закрытие с подтверждением при несохранённых изменениях

Файлы и экспорт

  • Меню Файл: создать, открыть, сохранить, сохранить как, экспорт HTML / DOCX / PDF
  • Tauri: нативные диалоги; браузер: File System Access API или download
  • Локальные картинки в десктопе: относительные пути к каталогу открытого .md

Настройки

  • Интерфейс — шрифты, размер, цвет и фон отдельно для редактора и предпросмотра
  • Контраст блоков кода (мягкий / обычный / высокий / максимальный)
  • Курсор редактора: цвет, форма/ширина, моргание
  • Панель инструментов — перетаскивание кнопок и разделителей, сброс раскладки
  • Проверить обновления — сравнение с GitHub Releases; в Windows можно скачать и запустить установщик
  • О программе — краткое окно с версией, copyright и ссылкой на GitHub

Форматирование

Жирный, курсив, зачёркивание, инлайн-код, выделение, индексы, H1–H4, списки и задачи, цитата, блок кода, HR, изображение, ссылка. Тогл форматирования и подсветка активных кнопок. Контекстное меню по правому клику.

Оглавление и поиск

  • Боковая панель TOC (H1–H4), вставка/обновление блока в документе
  • Поиск и замена со счётчиком совпадений

Горячие клавиши

Работают по event.code (не зависят от раскладки). На macOS — Cmd вместо Ctrl.

Где хранятся настройки

В localStorage браузера / WebView: тема, вид, оглавление, split, сессия вкладок, интерфейс (md-editor-interface), раскладка панели (md-editor-toolbar-layout), автопроверка обновлений (md-editor-update-auto).

Требования

  • Node.js 18+ (рекомендуется LTS)
  • Для десктопа (Tauri): Rust и зависимости Tauri для Windows

Установка и запуск

npm install

Веб (браузер):

npm run dev

Десктоп (Tauri):

npm run tauri:dev
