Возможности

Редактор и предпросмотр

  • Двухпанельный интерфейс: исходный Markdown и HTML-предпросмотр
  • Режимы отображения: обе панели / только редактор / только превью
  • Полный экран (F11)
  • Изменяемая граница между панелями (перетаскивание, клавиатура, двойной щелчок — 50/50)
  • Живой рендер через marked
  • Нумерация строк и подсветка текущей строки
  • Статус-бар: число слов, символов, позиция курсора
  • Копирование содержимого блоков кода в превью
  • Тёмная и светлая темы (сохраняются в localStorage)
  • Автосохранение каждые 3 секунды + восстановление сессии после перезагрузки

Вкладки

  • Несколько документов одновременно
  • Переключение вкладок с сохранением содержимого, курсора и прокрутки
  • Ctrl+Tab — быстрое переключение по последнему использованию (MRU)
  • Отдельная история undo/redo на каждую вкладку
  • Индикатор несохранённых изменений
  • Закрытие вкладки с подтверждением

Файлы

  • Открыть: диалог выбора .md / .markdown / .mdown / .txt
  • Сохранить (Ctrl+S): тихая перезапись открытого файла
  • Сохранить как… (Ctrl+Shift+S): диалог выбора пути
  • Tauri (десктоп): нативные диалоги и запись на диск
  • Браузер: File System Access API (Chrome / Edge) или fallback — скачивание

Экспорт

  • HTML — полноценный HTML-документ
  • DOCX — Word-документ (заголовки, списки, код, цитаты, ссылки, индексы, таблицы)
  • PDF — документ A4 (html2pdf.js), с встраиванием локальных изображений в десктопе

Форматирование

Панель инструментов: отмена/повтор, вырезать/копировать/вставить, жирный, курсив, зачёркивание, инлайн-код, выделение, верхний/нижний индекс, заголовки H1–H4, списки, списки задач, цитата, блок кода, горизонтальная линия, вставка изображения и ссылки.

Подсветка активных кнопок: если курсор внутри форматирования, соответствующая кнопка подсвечивается. Повторное нажатие снимает форматирование (тогл).

Автозакрытие скобок и кавычек: (), [], "", '', ``.

Оглавление (TOC)

  • Боковая панель структуры документа (H1–H4)
  • Клик по пункту — переход к заголовку
  • Подсветка текущего раздела по позиции курсора
  • Вставка / обновление блока TOC в документе

Поиск и замена

  • Найти далее (циклически)
  • Заменить текущее вхождение / заменить все
  • Счётчик совпадений N / M

Горячие клавиши

Работают по event.code (не зависят от раскладки клавиатуры). На macOS вместо Ctrl используется Cmd.

Ctrl+BЖирный
Ctrl+IКурсив
Ctrl+SСохранить
Ctrl+Shift+SСохранить как…
Ctrl+FФокус в поле поиска
Ctrl+HФокус в поле замены
Ctrl+TНовая вкладка
Ctrl+WЗакрыть вкладку
Ctrl+TabСледующая вкладка (MRU)
Ctrl+Shift+TabПредыдущая вкладка (MRU)
F11Полный экран
Ctrl+ZОтменить
Ctrl+Y / Ctrl+Shift+ZПовторить

Требования

  • Node.js 18+ (рекомендуется LTS)
  • Для десктопа (Tauri): Rust и зависимости Tauri для Windows

Установка и запуск

npm install

Веб (браузер):

npm run dev

Десктоп (Tauri):

npm run tauri:dev
