# Changelog

## 1.1.3 (2025-07-20)

### Added
- Подсветка синтаксиса в блоках кода (highlight.js, тема github-dark)
- Метка языка в левом верхнем углу блоков кода
- Контекстное меню по правому клику на выделенном тексте (форматирование, списки, блоки, медиа)
- Горячие клавиши `Alt+1`..`Alt+4` для заголовков H1–H4
- Перехват `Ctrl+N` и `Ctrl+Shift+T` в браузере (предотвращение стандартных действий)
- Скрипты `version:bump` и `version:sync` для управления версией

### Fixed
- Исправлен тогл форматирования при выделении справа налево (во всех format-функциях)
- Исправлено сохранение сессии при `beforeunload` и после сохранения файла на диск
- Перевод обработчика горячих клавиш в фазу захвата (`capture: true`)

### Changed
- Перегруппированы кнопки панели инструментов по логическим группам (inline, индексы, заголовки, списки, блоки)
- Обновлена документация (README, about.html)

---

## 1.1.2 (2025-07-18)

### Added
- Автосохранение каждые 3 секунды + восстановление сессии после перезагрузки
- Зачёркивание (`~~`) и выделение (`==`)
- Списки задач (`- [ ]`)
- Автозакрытие скобок и кавычек
- Переключение вкладок `Ctrl+Tab` (MRU)
- Полный экран (`F11`)
- Режимы отображения (обе панели / только редактор / только превью)
